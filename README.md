# Hastane-Projesi
Hastanın kendi üzerinde işlem yapabildiği,
Doktorun hem hasta üzerinde ve kendi bilgileri üzerinden işlem yapabildiği
Ve sekreterin hem hasta için randevu alımı, hasta kaydetme ve silme, doktor kaydetme ve silme, branş kaydetme ve silme yetkisine sahip
SQL Server ile database kapsamlı hastane projesi

Hasta Girişi İçin :
TC : 11111111111
Şifre : 1111

Doktor Girişi İçin : 
TC : 22222222222
Şifre : 2222

Sekreter Girişi İçin :
TC : 33333333333
Şifre : 3333
